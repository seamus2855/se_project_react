# WTWR (What to Wear?) — Final Project

## 🌦️ About the project
The idea of the application is simple yet powerful: we fetch real-time weather data via an API, process it, and recommend the most suitable clothing from the user's personal wardrobe based on the current forecast.

This project was built from the ground up using **React** for the frontend and **Express/MongoDB** for the backend, following a strict **Figma design** to ensure a high-quality UI/UX.

## 🚀 Features
- **Weather-Based Logic:** Recommends clothes based on temperature (Hot/Warm/Cold).
- **User Authentication:** Secure Sign-up, Login, and Profile editing.
- **Wardrobe Management:** Users can add, delete, and "like" clothing items.
- **Smart UI:** Optimized with a universal loading handler (`isLoading`) and custom hooks for modal management and form handling.
- **Responsive Design:** Fully styled using BEM methodology and Cabinet Grotesk typography.

## 🛠️ Tech Stack
- **Frontend:** React, React Router, Context API
- **Backend:** Node.js, Express.js, MongoDB, GCP (Google Cloud)
- **Design:** Figma
